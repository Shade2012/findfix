## Tentang FindFix

Findfix merupakan aplikasi web untuk pencatatan laporan barang baik temuan dan hilang, dan repo ini merupakan back-end yang digunakan dalam mengembangkan findfix, menggunakan mysql sebagai database, dan laravel.

## Setup Enviroment

- Buat file .env dan copas semua env.example ke file .env
- Jalankan migrasi dan seeder 'php artisan migrate:fresh --seed '
- Membuat Symlink 'php artisan storage:link'

## Cara Menjalankan Aplikasi

- Lakukan perintah 'php artisan serve' di terminal
